# Markdown 渲染

笔记本编辑器保存 Markdown 原文，预览区域将同一份原文转换为可阅读的文档显示。编辑区和预览区使用同一当前内容，保存后内容写入当前日报、周报或月报文件。

## 编辑区

编辑区支持普通文本输入、撤销、重做、剪贴板粘贴和图片插入。Markdown 源码保持原样保存，编辑器不会因为预览渲染而改写标题、列表、链接或强调符号。

Markdown 语法高亮是独立的显示功能。开启时只改变 Markdown 语法类别和符号的颜色，不改变字符、字体形状、保存文本或预览结果；关闭时编辑区显示普通文本。

## 预览内容

预览支持标题、段落、列表、链接、图片、表格、代码块和数学公式等 Markdown 内容。代码块会按照语言标记显示对应的代码块样式，数学公式使用 LaTeX 规则渲染。

预览区域支持文本选择和滚动。Markdown 链接可以打开对应链接，图片按照笔记所在目录和数据目录中的引用路径查找并显示。

## 图片引用

通过笔记本插入的图片会在 Markdown 中保存为图片链接，预览时根据当前笔记路径解析相对路径。删除 Markdown 图片链接只会改变正文引用，不会自动删除数据目录中的图片文件；图片文件的清理由存储管理页面处理。

## 空内容与异常

当前 Markdown 为空时，预览区域显示空状态提示，不会生成虚假的正文。图片路径无效时，正文中的链接仍然保留，但预览无法显示对应图片。无法识别的 Markdown 结构会按可解析的普通文本或基础结构显示，不会删除原始源码。
