# 搜索笔记

搜索笔记用于在当前选定的日报、周报或月报范围内筛选包含查询内容的文档。它只负责定位文档，不直接把搜索命中的正文全部展开；选中结果后，编辑器才读取对应文件的完整内容。

## 搜索范围

搜索范围由笔记本当前选择的类型决定：选择日报时只搜索日报目录，选择周报时只搜索周报目录，选择月报时只搜索月报目录。切换日报、周报、月报会清空并重新执行当前搜索。

## 匹配规则

查询内容会先去除首尾空白，至少需要两个字符。搜索按连续双字符匹配处理，支持中文和英文内容，不区分英文字母大小写。

匹配范围包括文档正文、文件名称和文档标题。正文索引保存用于筛选的搜索信息，不保存完整正文副本；打开结果时再从原 Markdown 文件读取正文。

## 结果规则

结果最多返回 100 条。笔记本搜索结果按文档名称倒序排列，不按命中行或命中次数改变顺序；文档名称通常对应日期、ISO 周或月份，因此列表优先显示较新的记录。

搜索结果会显示文档标题、文件名称和预览信息。预览用于列表识别，内容较长时会截断；点击结果后，右侧编辑区加载完整正文。

## 搜索时机

输入查询后会短暂等待再执行搜索，避免每个按键都立即扫描。查询变化、笔记保存或笔记类型切换会使旧搜索结果失效，只有与当前查询和当前类型一致的结果才会显示。

查询少于两个字符时，页面清空搜索结果，不发起有效匹配。没有命中时显示空结果状态，不会修改当前选中的 Markdown 内容。

## 索引更新

打开笔记本时会先显示可用的索引列表，并在后台检查目录变化。保存文档后会立即更新当前文件的索引；新增、删除或外部修改的文件会在索引刷新后反映到列表和搜索中。索引发生异常时，应用可以使用直接读取文件的方式作为兼容路径。
