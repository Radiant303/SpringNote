# 统计

统计页面用于查看 SpringNote 的使用记录、笔记数量、桌面组件计时结果和 AI 模型用量。页面中的数据分为两类：笔记数量会读取当前笔记目录中的文件，使用次数、计时收益和模型用量来自应用运行期间保存的统计记录。

## 统计范围

页面顶部可以选择统计范围：

- **全部**：从历史统计开始时间汇总到当天，并展示有数据的日期。
- **最近 30 天**：包含今天在内的最近 30 个自然日。
- **上个月**：统计上一个自然月的第一天至最后一天。
- **上个季度**：统计当前季度之前的完整自然季度。
- **自定义**：选择开始日期和结束日期，按两个日期之间的自然日统计。

开始日期和结束日期均包含在统计范围内。切换范围后，年度热力图仍保持最近一年的显示范围；总览和用量趋势会按照当前选中的范围重新汇总。

## 年度热力图

年度热力图显示最近一年每天的活跃次数，日期从左到右按周排列，纵向按星期排列。颜色深浅表示当天的活跃次数：

- 0 次：无活跃记录；
- 1–2 次：低活跃；
- 3–4 次：中等活跃；
- 5–7 次：较高活跃；
- 8 次及以上：高活跃。

活跃次数来自首页内容整理成功事件和编辑补全成功事件。应用启动、打开页面、浏览已有笔记以及单纯启动桌面组件不会增加热力图次数。鼠标悬浮日期单元格时，会显示该日期和对应的活跃次数。

热力图统计的是应用活动记录，不是日报文件数量，也不是输入字数或 AI Token 数量。某天存在日报但没有产生统计事件时，该日期仍可能显示为零。

## 总览

总览中的每个数字都对应当前选定的时间范围，数据含义如下：

### 总结数

表示首页内容整理成功的次数。每天最多计入 10 次，超过 10 次的生成不会继续增加该指标。统计的是成功事件，不是首页展示过的栏目数量，也不是日报文件数量。

### 编辑补全次数

表示 AI 编辑补全成功的次数。请求失败、取消或没有产生有效补全内容时不会增加该指标。

### 总记录数

表示统计范围内日报、周报和月报数量的合计。该数字是三个分类数量相加，不代表 Markdown 文件总数之外的其他输入记录。

### 日报数、周报数、月报数

分别表示当前统计范围内符合对应文件规则的 Markdown 笔记数量。日报按文件日期统计；周报和月报按其覆盖的日期范围统计，因此跨越统计范围的周报或月报，只要覆盖范围内的日期，就可能被计入。

删除或新增笔记文件后，重新加载统计页面才会重新读取文件数量。文件内容是否为空不会改变文件数量统计，只要文件符合对应类型和日期规则即可计入。

### 输入 Tokens

表示模型请求发送给供应商的输入 Token 数量，通常包括提示词、上下文和本次输入内容。它反映请求规模，不等同于用户输入的字数。

### 输出 Tokens

表示模型返回内容产生的输出 Token 数量。流式响应会在请求结束并获得用量信息后计入统计。

### 缓存 Tokens

表示供应商返回的缓存输入 Token 数量。供应商未提供缓存用量时，该部分为零；零不表示请求失败。

### 应用启动次数

表示应用启动事件次数。选择全部范围时会同时兼容历史版本保存的启动计数；选择有限日期范围时，只统计能够对应到日期的启动记录。

### 工作时长与收益

工作时长来自桌面组件运行期间累计的计时，收益来自同一计时过程按当前配置换算出的数值。暂停、停止或应用退出时会提交已累计的部分；异常退出前尚未提交的短暂计时可能不会进入统计。

收益是应用内的估算指标，不代表工资结算或真实收入。修改时薪等配置只影响之后的计时换算，不会重算已经保存的历史统计。

## 用量趋势

用量趋势按日期显示当前范围内的模型 Token 用量。每天的柱形高度按照当天总 Token 数相对当前图表最大值缩放，柱形内部按供应商分段显示。

鼠标悬浮有用量的日期时，可以查看：

- 日期；
- 当天总 Token 数；
- 输入、输出和缓存 Token 明细，或供应商 Token 明细；
- 主要供应商各自占用的 Token 数量。

图例最多显示用量最高的 8 个供应商，其余供应商合并为“其他”。供应商用量按供应商名称和模型调用日期汇总；同一供应商下的多个模型会合并显示在该供应商的分段中。

没有模型调用记录时，趋势区域显示为空状态。某天没有 Token 用量时不会生成可见柱形，但日期仍会按照所选范围排列。全部范围下，图表只展开实际存在用量记录的日期；有限范围下，会保留范围内的完整日期轴。

## 数据更新与异常

统计记录在事件发生后写入本地数据目录。首页整理、编辑补全、应用启动、桌面组件计时和模型请求分别记录，某一类记录失败不会阻止其他功能继续工作。

如果统计文件读取失败，统计页面会显示空统计结果，不会阻止笔记编辑、应用启动或 AI 请求。此时页面中的零值表示当前读取没有得到统计数据，不代表历史数据已经被删除。

模型供应商未返回 Token 用量时，该请求可能无法出现在用量趋势中；请求失败不会计入成功的首页整理或编辑补全次数。笔记文件读取失败时，笔记数量可能暂时显示为零，但不会删除实际笔记文件。
