# 更新与版本

SpringNote 会持续优化功能体验、修复问题并提升稳定性。升级前建议查看对应版本更新日志，了解新增功能、行为变化以及可能需要重新确认的配置。

## 查看当前版本

打开`设置 > 关于`即可查看当前版本、系统信息以及项目链接。

## 安装更新

请前往项目[发布页面](https://github.com/Radiant303/SpringNote/releases/latest)页面或者[QQ交流群](https://qm.qq.com/q/c6QiowtYSA)下载与你的平台和系统架构匹配的安装包。

更新前建议：

1. 退出正在运行的 SpringNote。
2. 如果正在使用桌面组件、托盘功能或同步功能，请等待相关操作完成。
3. 根据自己的存储策略定期备份重要数据。


## 更新后检查

首次启动新版时，应用可能需要检查已有数据或初始化新功能。

进入主界面后，建议确认：

1. 数据目录是否正确。
2. 最近的日报和图片是否能够正常打开。
3. 供应商配置和默认模型是否仍然存在。
4. 快捷键、托盘以及桌面组件是否符合预期。

如果升级后部分 AI 功能不可用，请优先检查：

- 供应商连接状态
- 默认模型配置
- 网络连接情况

不要直接删除原有配置。

## v1.0.3（2026-07-10）

### 深色模式、壁纸与 Markdown 编辑增强

#### 界面优化

- 优化便签编辑页右上角「已保存」与编辑预测提示的切换显示。
- 优化 Markdown 渲染效果，改善标题、表格、公式、任务列表等内容展示。
- 优化官网页面样式，完善配置教程。

#### 功能新增

- 新增深色模式支持，并适配 Windows/macOS 桌面组件。
- 新增应用及桌面组件壁纸功能，支持：
  - 默认背景
  - 透明度调整
  - 模糊效果
  - 蒙版配置

- 新增便签页编辑模式、分栏模式和预览模式。
- Markdown 编辑器新增语法高亮功能，可在设置中开启或关闭，默认启用。
- Markdown 渲染中的链接支持点击后使用浏览器打开。
- 快捷键设置支持直接录入组合键，并支持重置和清除全局快捷键。
- 新增存储管理功能，可扫描未被便签引用的图片并在确认后清理。

#### 问题修复

- 修复便签自动同步时上传对象遗漏的问题。
- 修复回忆书本地降级检索中的工具调用上下文格式不兼容问题。
- 修复 Markdown 表格、嵌套方括号以及部分链接解析异常。
- 修复桌面组件高频拖动时状态循环、纯色切换异常以及字体缩放下控件边界异常。

#### 问题修复

- 修复 Windows 多次启动无法复用已有窗口的问题。
- 修复便签搜索无法匹配完整内容的问题。
- 修复百炼官方 DeepSeek 无法调用工具的问题。
- 修复 AI 调用统计写入不完整的问题。
- 修复启动时配置文件损坏导致应用崩溃的问题。
- 修复桌面组件球形模式快速移动显示异常的问题。
- 修复跨平台网络权限声明不一致的问题。
