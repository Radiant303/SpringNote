# 快捷键

快捷键用于在不切换窗口的情况下打开 SpringNote、隐藏窗口或快速执行输入操作。快捷键设置支持 Windows 和 macOS，录入方式会根据当前平台显示对应名称。

## 全局快捷键

全局快捷键在 SpringNote 不处于前台时也可以使用，常用于快速显示或隐藏窗口。

1. 打开设置中的快捷键页面。
2. 点击全局快捷键的录入区域。
3. 按下想使用的组合键。
4. 检查显示的组合键并保存。
5. 关闭窗口后实际测试一次。

录入区域接收真实按键，并识别平台修饰键和快捷键冲突。按键名称会根据当前系统显示为对应的格式。

## 输入快捷键

输入快捷键用于快速进入记录流程或执行输入相关操作。录入后，按下组合键即可触发对应功能；如果组合与系统或其他应用冲突，页面会提示更换。

Windows 使用 Ctrl、Alt、Win 等平台按键名称；macOS 使用 Command、Option、Control 等名称。应用按照当前系统显示对应名称。

## 无法触发时

先确认 SpringNote 仍在运行且没有退出托盘，再确认快捷键没有被其他软件占用。修改后重新保存，并用一个不常见的组合测试。
