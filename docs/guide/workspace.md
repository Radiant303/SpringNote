# 工作区模式

工作区模式决定笔记本右侧编辑区域如何呈现当前 Markdown 文档。它包含编辑、分栏和预览三种状态，三种状态共用当前选中文档和当前内容。

## 编辑

编辑模式显示 Markdown 源码，光标、选区、撤销、重做、AI 实时补全和自动保存均在编辑区域内进行。文本变化会写入当前选中的文件，并同步更新列表信息。

## 分栏

分栏模式同时显示 Markdown 编辑区和渲染预览区。编辑区的变化会实时传递给预览区；预览区用于查看最终显示效果，不单独产生另一份文档内容。

## 预览

预览模式隐藏源码编辑区，只显示当前 Markdown 的渲染结果。预览支持选择文本、滚动、链接、图片、代码块、表格和数学公式；返回编辑模式后继续使用同一份源码。

## 切换与保存

编辑、分栏和预览之间的切换只改变工作区布局，不改变 Markdown 内容、当前文档或保存状态。模式选择会写入配置文件，重新启动应用后恢复上一次选择。

切换文档时，工作区模式保持不变；新文档加载完成后，编辑区和预览区显示新文档内容。正在保存或加载文档时，页面会暂时限制对应操作，避免旧文档内容覆盖新选择。
