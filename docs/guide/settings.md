# 偏好设置

偏好设置页面由八个独立卡片组成。每个卡片只修改对应配置，不会改变其他卡片的设置；修改后的配置会保存到应用配置文件，并由相关功能读取。桌面组件相关内容单独归入“组件”分组。

## 个人信息

个人信息包含每日工作时长、日薪和所在行业。每日工作时长和日薪用于桌面组件的收益计时，所在行业作为 AI 整理时的背景信息。

每日工作时长单位为小时，默认值为 8；日薪单位为人民币，默认值为 200。收益计算使用：

```text
每秒收益 = 日薪 ÷ (每日工作时长 × 3600)
```

每日工作时长小于或等于 0 时，计算按 8 小时处理。组件运行时收益逐秒累加，暂停后停止累加；跨本地日期后，当天计时和当天收益重新开始，累计统计继续保留。

所在行业默认值为“互联网”。它会作为日报整理等 AI 请求的上下文，用于调整术语和表达，不会改变原始事实，也不会单独触发 AI 请求。修改个人信息不会重新生成已有文档。

## 字体与显示

字体与显示包含应用字体、主题模式、字体大小和 Markdown 语法高亮四项设置。

**应用字体**决定界面和编辑器使用的字体。选择系统字体时使用当前平台的默认字体；选择其他可用字体时，界面和编辑区域会在下一次绘制时使用该字体。字体选择只影响显示，不改写文档内容。

**主题模式**有跟随系统、浅色和深色三种状态。跟随系统时，应用根据操作系统当前主题切换；浅色和深色会覆盖系统主题。主题切换只改变颜色、背景和控件外观，不改变壁纸文件或文档。

**字体大小**使用百分比表示，默认值为 100%，可调整范围为 80% 到 140%。它会影响应用界面和编辑器的字号，布局会随字号重新计算；不会改变 Markdown 文件中的文字大小。

**Markdown 语法高亮**默认启用。启用时，编辑器只对 Markdown 符号和语法类别改变颜色；关闭时显示普通文字。无论开关状态如何，字符、符号形状、保存文本和预览渲染结果都不改变。

## 行为与启动

行为与启动包含开机自启动、显示更新和 API 网络日志。

**开机自启动**开启后，系统登录时启动 SpringNote。该选项只在支持的平台上可用；不支持的平台会显示平台限制。

**显示更新**控制应用是否显示版本更新提示。关闭后不影响手动查看版本和更新日志。

**记录 API 网络日志**开启后，会记录 AI 请求相关的网络日志，用于排查供应商连接、请求参数和返回错误。它不等同于开启 AI，也不会改变请求内容；日志可能包含请求元信息，使用时应注意日志文件的保存和共享范围。

## 壁纸

壁纸卡片只控制主应用窗口的背景。模式包括默认背景、本地图片和纯色。

选择本地图片后，图片会复制到数据目录的壁纸位置，原始图片不会被删除。填充方式包括拉伸、覆盖和居中：拉伸会改变图片比例，覆盖会保持比例并裁剪超出窗口的部分，居中会保持原尺寸并在剩余区域显示底色。

图片和纯色模式都支持不透明度、模糊度和蒙版浓度。模糊度最高为 25；蒙版用于在背景上增加统一色层。不透明度、模糊度和蒙版只改变主窗口的背景显示。

透明控件模式开启后，卡片和侧边栏会减少自身遮挡，让壁纸透过控件显示。该模式下还可以调整控件不透明度、是否保留卡片描边以及文字颜色加深；关闭透明控件模式时，这三个联动选项不参与显示。

恢复默认会将主窗口壁纸模式、图片、颜色、透明度、模糊和控件显示恢复为默认状态。

## 托盘

托盘卡片包含显示托盘图标和关闭时最小化到托盘两个开关。

关闭托盘图标后，关闭时最小化到托盘会同时失效。关闭主窗口和退出应用的行为不同：最小化到托盘表示进程继续运行，可以通过托盘重新打开；退出应用才会结束后台进程。

托盘功能只在支持的平台上启用，不支持的平台会显示平台限制。

## 数据保存

数据保存卡片显示当前数据目录，并通过文件夹选择器更换目录。数据目录保存日报、周报、月报、图片和应用配置；首页选择的普通文件目前不会作为附件复制到这里。

更换目录会执行数据迁移并更新应用使用的位置。迁移成功后，后续记录和图片写入新目录；原目录不会因为切换而自动作为当前目录继续使用。迁移过程中如果出现错误，当前配置不会切换到未确认成功的位置。

## 提示词

提示词卡片包含首页栏目和日报整理提示词。

首页栏目用于编辑三个栏目的标题和 AI 说明。标题用于首页显示；AI 说明用于告诉 AI 每个栏目关注的内容；AI 说明为空时使用对应标题作为说明。保存后影响后续整理，不会重新处理已有首页概览。

日报整理提示词用于控制 AI 如何把已有日报和新增输入合并成当天 Markdown。提示词可使用当前日期、已有日报、新增记录和所在行业等内置信息。修改后只影响后续日报整理，不改变已经保存的日报。

## 回忆书检索

回忆书检索卡片控制回忆书工具读取和返回内容的数量与长度：

| 设置 | 作用 | 默认值 | 范围 |
| --- | --- | ---: | ---: |
| 回忆书单轮最大搜索次数 | 单轮回答中允许执行的搜索次数 | 12 | 1–120 |
| 单条结果返回最大字符数 | 单条记录返回给回答流程的最大字符数 | 3600 | 80–10000 |
| 连续日报读取最大数量 | 一次读取连续日报时允许读取的最大条数 | 31 | 1–31 |
| 关键词搜索结果最大数量 | 关键词搜索返回的最大记录数 | 12 | 1–200 |
| 命中关键词截取前最大字符数 | 命中位置前保留的上下文字符数 | 1400 | 0–4000 |
| 命中关键词截取后最大字符数 | 命中位置后保留的上下文字符数 | 2600 | 0–6000 |

这些设置只影响回忆书检索返回给 AI 的内容规模，不会修改本地 Markdown 文件，也不会改变笔记本搜索的返回上限。
