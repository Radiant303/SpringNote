# 快速入门

SpringNote 的功能由记录、整理和回顾三部分组成。AI 配置不是保存普通记录的前置条件。

## 第一步：确认数据位置

首次使用时先确认数据保存目录。日报、周报、月报、图片和相关配置都会围绕这个目录保存；当前选择的普通文件不会复制进数据目录。

## 第二步：保存第一条记录

1. 回到首页。
2. 在快速输入框输入文字。
3. 需要时粘贴图片或添加文件。
4. 点击提交。

提交后，输入会参与当天日报生成，图片会保存并插入可预览链接；文件目前只把文件名和路径加入本次输入，不会作为附件复制到数据目录。日报内容由 AI 合并结果或本地合并结果决定，不保证额外保留未经处理的原始输入副本。

## 第三步：整理日报

打开笔记本并选择日报，在编辑模式补充上下文；分栏和预览模式用于查看 Markdown 渲染结果。日报保存具体事实和过程，周报、月报保存周期性整理结果。

## 第四步：配置 AI

在设置中添加供应商，测试连接后获取或手动添加模型，再分别选择智能生成、编辑补全和回忆书模型。三种用途可以使用不同模型。

配置完成后，可以在首页使用智能生成，在笔记本使用实时补全，在回忆书中搜索并询问历史记录。

## 第五步：建立回顾习惯

每天用日报保存事实，每周整理周报，每月整理月报。遇到“我什么时候做过某件事”或“某段时间发生了什么”时，在回忆书中提问，而不是手动翻找所有文件。
