# 快速入门

>SpringNote 核心功能由记录、整理和回顾三部分组成

## 第一步：确认数据位置

首次使用时先确认数据保存目录。日报、周报、月报、图片和相关配置都会围绕这个目录保存；当前选择的普通文件不会复制进数据目录。

![数据目录](/images/datadir.png)


## 第二步：配置 AI


以 **DeepSeek** 为例进行配置说明：

#### ① 添加供应商 BaseURL请填写 https://api.deepseek.com/beta


>
> 此处填写`beta`原因是Deepseek的[FIM接口要求](https://api-docs.deepseek.com/zh-cn/guides/fim_completion)
>
>其他的OpenAI兼容接口请依据实际情况填写
>

#### ② 手动添加模型 deepseek-v4-flash

>
>因为Deepseek的`beta`接口不支持模型列表查询，所以需要手动添加模型

>请手动勾选补全类型

## 第三步：完成第一次记录

![首页](/images/index.png)
### 在首页输入当天内容

打开首页，在快速输入框中输入当天的工作记录、想法或待处理事项。文字内容会参与当天日报保存和首页三栏整理。

图片可以通过图片按钮选择，也可以从剪贴板粘贴。图片提交后会保存到数据目录，并在日报中生成可预览的 Markdown 图片链接。

文件按钮目前只把文件名称和路径加入本次输入。文件不会被复制到数据目录，应用也不会读取文件内容；路径信息不代表 AI 已经解析了文件。



### 执行智能生成

输入内容后点击“智能生成”。如果已配置可用的智能生成模型，应用会根据提示词设置整理首页三栏，并尝试将新增内容合并到当天日报。

如果没有配置模型，或模型请求失败，文字和已保存的图片仍然可以保存，日报会使用本地合并结果。智能生成不是保存普通记录的前置条件。

首页三栏显示当天整理结果。点击栏目卡片可以打开详情弹窗，查看该栏目中的完整条目。

## 第四步：在笔记本中查看和编辑

![笔记本](/images/note.png)

进入“笔记本”后，可以在日报、周报和月报之间切换。日报按自然日保存，周报按 ISO 周保存，月报按自然月保存。

选中记录后，编辑区会加载 Markdown 原文。编辑内容会自动保存，保存后列表中的标题、预览和搜索结果会同步更新。右上角可以切换“编辑”“分栏”和“预览”模式：

- **编辑**：直接修改 Markdown 源文；
- **分栏**：同时查看 Markdown 源文和渲染结果；
- **预览**：只查看渲染后的内容。

笔记本搜索只搜索当前选择的日报、周报或月报类型。搜索至少输入两个字符，结果最多返回 100 条；点击结果后可以打开对应的完整正文。

## 第五步：使用回忆书

![回忆书](/images/memories.png)

进入“回忆书”后，可以直接询问已经保存的工作记录。回忆书会根据问题决定执行关键词搜索、按日期读取日报、读取周报或读取月报，再根据取得的内容生成回答。

没有配置回忆书模型时，关键词搜索和记录读取仍然可用，但不会生成 AI 回答。回忆书不会自动修改日报、周报或月报。

回忆书顶部的思考模式包含“关闭”“High”和“Max”。该设置只影响 AI 回答请求，不影响本地搜索和记录读取。

点击右上角“开启新对话”会清空当前回忆书会话消息，并从空的上下文开始。该操作不会删除本地笔记、搜索索引或回忆书配置；当前应用不提供旧会话历史恢复功能。

## 第六步：使用牛马时钟

![牛马时钟](/images/components.png)

在 Windows 或 macOS 上，SpringNote 可以显示独立的桌面组件。组件用于查看当前计时、当天工作时长和收益，并在主窗口之外控制计时。

- 左键单击组件：开始或暂停计时；
- 右键单击组件：打开主窗口并进入首页；
- 左键拖动组件：移动窗口位置；
- 圆球模式下将鼠标移入组件：展开组件；
- 鼠标离开圆球有效区域：收起组件。

组件位置会在拖动后保存，多显示器环境下会限制在可见工作区内。计时收益由设置中的每日工作时长和日薪换算，修改设置不会重算历史收益。

## 第七步：配置常用功能

完成基本记录后，可以在设置中继续配置：

- **偏好设置**：个人信息、字体与显示、行为与启动、壁纸、托盘、数据保存、提示词和回忆书检索；
- **供应商**：添加或关闭 AI 服务，测试连接并维护模型列表；
- **默认模型**：分别选择智能生成、编辑补全和回忆书使用的模型；
- **快捷键**：通过实际按键录入全局快捷键和输入快捷键；
- **云同步**：配置 WebDAV 连接和同步时机；
- **存储管理**：扫描、预览并清理不再被笔记引用的图片；
- **统计**：查看热力图、笔记数量、AI 用量、工作时长和收益。
