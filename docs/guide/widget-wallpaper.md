# 组件壁纸

组件壁纸只控制桌面组件窗口的背景显示，与主应用壁纸完全独立。它不会改变组件中的计时、收益、按钮和文字内容，也不会影响主窗口的背景。

## 背景模式

组件支持以下背景模式：

- **默认背景**：使用组件的默认白色背景；
- **纯色**：使用选择的背景颜色；
- **本地图片**：使用选择的图片作为组件背景。

切换背景模式只改变组件窗口的显示，不会修改计时状态、位置或笔记内容。

## 纯色背景

纯色模式可以选择背景颜色。颜色只作用于组件背景层，组件文字、计时按钮和交互状态仍由组件主题决定。

## 图片背景

选择本地图片后，图片会复制到应用数据目录的组件壁纸位置，原始图片不会被删除。组件读取复制后的文件显示背景，原始文件之后发生变化不会自动改变已经保存的组件壁纸副本。

图片路径失效或文件无法读取时，组件会使用可用的默认背景显示，不会删除计时记录或修改组件位置。

## 不透明度

非默认背景支持调整背景不透明度。该设置只改变背景层的透明程度，组件内容层仍保持可读性；默认白色背景不显示该调节项。

不透明度变化不会改变图片文件本身，也不会影响主应用壁纸的不透明度或透明控件设置。

## 恢复默认

恢复默认会清除组件壁纸模式、颜色、图片路径和不透明度等自定义背景设置，使组件回到默认背景。

恢复默认不会删除数据目录中的壁纸文件，不会修改主应用壁纸，不会改变组件的显示开关、圆球模式、位置、计时状态和收益统计。
