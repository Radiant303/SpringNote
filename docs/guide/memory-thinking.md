# AI 思考模式

AI 思考模式控制回忆书回答前使用的分析等级。它只影响发送给模型的思考或推理参数，不改变本地关键词搜索、记录读取、搜索范围和 Markdown 文件。

## 可选等级

- **关闭**：不请求额外的思考过程，仍然可以正常生成回答。
- **High**：请求较高等级的思考或推理处理。
- **Max**：请求当前协议支持的最高等级。

思考等级可能影响模型处理时间和 Token 用量。最终是否返回可见的思考内容，取决于供应商协议、模型能力和服务端响应格式。

## 协议映射

应用会根据供应商使用的请求协议转换界面等级，不会把界面名称直接发送给所有服务：

| 回忆书界面等级 | 支持思考等级参数的服务 | 支持推理等级参数的服务 |
| --- | --- | --- |
| 关闭 | 关闭思考 | 不发送推理等级，并使用普通回答参数 |
| High | `high` | `high` |
| Max | `max` | `xhigh` |

这里的 `Max` 是应用界面等级，不代表所有供应商都使用同一个参数名称。聊天协议使用 `max`，Responses 协议使用 `xhigh`。

## 不支持或请求失败

供应商不支持所选思考级别、模型不接受对应参数或请求参数与协议不匹配时，当前 AI 请求会返回错误。已有搜索结果、已经读取的本地笔记和历史对话不受影响。

关闭思考模式不能绕过供应商认证、模型不存在、网络失败或接口协议错误；这些问题仍会导致当前请求失败。
