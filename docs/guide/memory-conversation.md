# 开启新对话

开启新对话会清空当前回忆书会话，并从空的消息上下文开始。新对话不会继续使用上一段对话中的问题、回答、思考内容和已经读取的记录。

## 上下文隔离

新对话不会继承上一段对话临时形成的检索结果和 AI 回答上下文。后续提问会重新判断需要使用的关键词搜索、日期读取或周期读取工具。

本地日报、周报和月报不会因为开启新对话而改变。新对话只影响回忆书会话消息，不会清空搜索索引、删除笔记或重置回忆书检索设置。

## 会话清理

开启新对话会清除当前已保存的回忆书消息。应用当前没有独立的历史会话列表，清除后无法在回忆书页面恢复上一段对话内容。

开启新对话前的本地笔记、搜索索引、供应商配置、思考模式和检索设置仍然保留。

## 模型状态

开启新对话不会重新选择回忆书模型，也不会改变当前思考模式和供应商配置。没有可用模型时，仍可在新对话中执行本地关键词搜索和记录读取；AI 回答能力仍取决于模型是否可用。
