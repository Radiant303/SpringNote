# 系统要求

## 支持的平台

SpringNote 提供 Windows 和 macOS 桌面版本。请下载与系统和处理器架构匹配的安装包；不匹配的版本可能无法启动或无法使用全部系统能力。

## Windows

Windows 版本要求为 64 位，系统需要处于受支持的更新状态。安装程序会将应用文件和运行所需组件放在安装目录中；如果系统缺少必要运行组件，安装或首次启动时可能提示补充安装。

桌面组件、托盘、全局快捷键和多显示器位置保存依赖 Windows 桌面环境。远程桌面、窗口管理工具或系统策略可能改变这些功能的表现。

## macOS

请使用与 Mac 处理器架构匹配的应用包。第一次打开从互联网下载的应用时，macOS 可能要求确认来源；这是系统安全机制，不代表应用数据异常。

## 网络与 AI

记录、编辑、Markdown 预览和本地搜索不要求 AI 网络连接。智能生成、实时补全和回忆书的 AI 回答需要网络、可用供应商和已选择的模型。

## 存储空间

除了应用本身，还需要为日报、周报、月报和图片预留空间。当前选择的普通文件不会复制到数据目录，因此不会因为文件选择直接增加应用数据占用。
