# Desktop Experience

SpringNote is designed for daily desktop use: tray behavior, global hotkeys, auto start, font settings, and the desktop status widget live in Settings.

## Desktop Status Widget

The widget shows level, income, work duration, and running state. It can expand into a card or collapse into a compact orb.

<div class="product-shot">
  <img src="/images/components.png" alt="Desktop status widget">
</div>

## Position Memory And Multi-Monitor Bounds

The widget remembers its last position and restores it after restart. In multi-monitor setups, it stays inside the visible desktop area.
