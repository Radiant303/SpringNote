# Model Setup

SpringNote supports multiple providers, model types, and default model assignments. Intelligent generation, edit completion, and the memory book can use different models.

## Setup Flow

      <p>Start with the provider name and Base URL. SpringNote uses providers as model groups, so multiple OpenAI-compatible services can stay organized.</p>
    </div>

  <section>

      <span>02</span>

      <p>Enter the model ID and mark what it can do. Different models can be assigned to generation, completion, or memory retrieval.</p>

  </section>

  <section>
    <div>
      <span>03</span>

      <p>Review the protocol, model type, and completion mode. These options define where the model appears across editing, summaries, and retrieval.</p>
