# Shortcuts

Shortcuts allow you to open SpringNote, hide the window, or quickly perform input operations without switching windows. Shortcut settings support Windows and macOS; key names are displayed according to the current platform.

## Global Shortcuts

Global shortcuts work even when SpringNote is not in the foreground, commonly used for quickly showing or hiding the window.

1. Open the Shortcuts page in Settings.
2. Click the global shortcut recording area.
3. Press the desired key combination.
4. Check the displayed combination and save.
5. Test it after closing the window.

The recording area captures actual key presses and identifies platform modifier keys and shortcut conflicts. Key names are displayed in the format appropriate for the current system.

## Input Shortcuts

Input shortcuts are used for quickly entering the recording flow or performing input-related operations. After recording, pressing the key combination triggers the corresponding function. If the combination conflicts with the system or other applications, the page prompts you to change it.

Windows uses platform key names such as Ctrl, Alt, Win; macOS uses names such as Command, Option, Control. The application displays the corresponding names according to the current system.

## When Shortcuts Don't Work

First confirm that SpringNote is still running and has not exited to the tray. Then verify that the shortcut is not occupied by other software. After modifying, save the change and test with an uncommon key combination.
