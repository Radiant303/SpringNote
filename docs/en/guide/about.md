# About

The About page displays the current version, system information, update entry, project links, and license information.

When encountering issues, first note the current version and steps to reproduce, then check the changelog to see if a fix already exists. Upgrading the application does not actively delete daily notes, images, or configuration from the data directory.

Of course, whether you encounter issues during use or have new ideas and suggestions, we welcome your feedback.

We take every piece of feedback seriously and continuously improve SpringNote.
