# 桌面体验

SpringNote 的桌面端围绕长期使用设计：托盘、全局快捷键、开机自启动、字体设置和桌面状态组件都可以在设置页管理。

## 桌面状态组件

桌面组件显示等级、收益、工作时长和运行状态。它可以展开为卡片，也可以收起为紧凑圆球。

## 位置记忆与多屏幕

组件会记住上次位置，重启后恢复。多屏幕环境下会限制在可见桌面区域内，避免窗口被拖出屏幕。
